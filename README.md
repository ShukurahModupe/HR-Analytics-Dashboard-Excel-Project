# HR Analytics Dashboard (Excel Capstone Project)

## 📌 Project Overview
This project is an HR Analytics Dashboard built in Microsoft Excel to analyze employee data and provide insights into workforce trends, attrition, compensation, and performance.

The project involved:
- Data Cleaning and Transformation
- Data Analysis using Pivot Tables
- Dashboard Creation and Visualization
- Interactive Filtering using Slicers

---

## 📊 Dashboard Metrics
The dashboard provides insights on:

- Total Employees
- Average Salary
- Overall Attrition Rate
- Total Bonus Payout
- Employee Count by Department
- Attrition by Department
- Bonus Payout by Performance Band
- Average Salary by Department

---

## 🛠 Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Lookup Functions (VLOOKUP/XLOOKUP)
- Data Cleaning Functions (TRIM, UPPER, IF, etc.)

---

## 📈 Key Insights
- Information Technology has the highest employee count.
- The organization has an attrition rate of 16%.
- Bonus payouts vary significantly across performance bands.
- Salary distribution differs across departments.
